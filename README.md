# roguelike-tds-continue
Рогалик, вдохновлённый Nuclear Throne.  
Сделано:
- один персонаж
- несколько оружий: винтовка, пистолет, дробовик, пулемёт
- несколько противников
- один босс
- один вид локаций (процедурная генерация)
- сундуки с оружим и боеприпасами
